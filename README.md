# Gismeteo Parser

Этот проект предназначен для парсинга структуры стран, регионов, районов и городов с сайта gismeteo.ru.

## Назначение
- Получение иерархической структуры населённых пунктов для выбранной страны.
- Экспорт результатов в JSON и TXT форматы.

## Быстрый старт
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите основной скрипт:
   ```bash
   python main.py
   ```
3. Следуйте инструкциям в консоли для выбора страны и получения результатов.

## Основные файлы
- `main.py` — основной скрипт.
- `countries.py` — справочник стран и их структур.
- `country_levels_list.txt` — текстовый файл с полным списком стран.
- `requirements.txt` — зависимости проекта.

## Особенности
- Поддержка разных структур стран (города, районы, регионы).
- Сохранение результатов в JSON и TXT.
- Обработка ошибок и информативные сообщения.
- Логирование событий и ошибок (файл gismeteo.log и консоль).
- Модульные тесты для основных функций парсера.

## Улучшения
- Аннотации типов для функций.
- Прогресс-бар для долгих операций (tqdm).
- Гибкая настройка и расширяемость.

## Тестирование
Для запуска модульных тестов используйте команду:

```
python -m unittest discover -s tests
```
